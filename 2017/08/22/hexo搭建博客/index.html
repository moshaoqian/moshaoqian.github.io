遇到的问题

在安装完nodejs之后执行sudo npm install -g hexo安装hexo一直报错，查阅资料之后原因是：“出现以上错误是因为权限问题。”将上面命令替换为sudo npm install --unsafe-perm --verbose -g hexo就好了。
